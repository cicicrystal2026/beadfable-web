# BeadFable V1 产品需求文档（PRD）

- 版本：V1.0
- 日期：2026-07-04
- 状态：已确认（基于三轮需求梳理对话收敛）
- 品牌 / 域名：BeadFable（beadfable.com，Cloudflare Workers 部署）

---

## 1. 一句话定位

**面向海外拼豆爱好者的「Q 版拼豆图纸 + 精确到颗的物料清单」内容交易站**：
社媒话题内容引流 → 网站承接转化 → 联盟链接 + 套装预售变现。

这不是一个工具站，而是一个**话题营销驱动的交易站**。转换器是引擎，Q 版惊喜感是门面，
「按图配料」是收银台。

---

## 2. 调研结论摘要

### 2.1 技术可行性
- 「图片 → 带色卡的拼豆图」核心环节是**经典算法**（缩放 + 颜色量化 + 最近色映射到真实
  豆子色卡），纯前端 JS 可实现，零边际成本，结果精确可控。
- 生成式 AI 的位置在**前置风格化**（照片 → Q 版化），可用 API：Google Nano Banana、
  OpenAI gpt-image、Retro Diffusion。直接像素化真人照片效果差，Q 版化后再像素化是
  效果好的关键。
- 纯工具赛道已拥挤（beadpattern.net、Somake、PixelPerler、pixelbeads.org、Beadify、
  拼豆星球、像素星球等 10+ 竞品），工具本身不构成壁垒。**竞品的共同断点：停在
  「导出图纸」，没有做「按图配料」的闭环。**

### 2.2 市场
- 全球熔融豆市场约 6.6 亿美元（2026），预计 2035 年 10.7 亿美元；Perler 约占 24% 份额，
  与 Hama、Artkal 三分海外市场。
- 中国供给端：义乌产业带日发货约 25 亿颗；淘宝拼豆搜索量同比 +500%。
- **最肥的一环是跨境零售/套装**：国内 39 元的货海外卖 27 美元，5~7 倍溢价。
- 纯图纸工具付费天花板矮（竞品订阅价 $14.99/年量级）；亚马逊联盟玩具类佣金仅 ~3%。

## 3. 关键决策记录（对话中已裁决）

| # | 决策 | 结论 |
|---|------|------|
| D1 | 产品叙事 | **只做 Q 版**。万能转换是引擎，Q 版是唯一门面。落地页第一屏卖「把你/你爱豆/你的猫变成 Q 版拼豆」，不卖「上传任何图片」。其他风格后续版本再扩。 |
| D2 | IP 合规路径 | **卖工具不卖受版权保护的形象**。预制图纸库全部为 AI 协助下的原创设计；营销「蹭话题不蹭形象」（如「适合 XX 粉丝的配色」）。用户自行上传版权图片属用户行为（打印店逻辑）。 |
| D3 | V1 交付范围 | **图纸 + 精确到颗的色号清单（BOM）+ 一键购买联盟链接**，同时放「为这张图配好豆子包 · $19.9 预售」按钮，用真实点击/下单意向验证需求，再决定是否投入自营电商。 |
| D4 | 目标市场 | 海外（英文站），以女性向社区为主的社媒引流：TikTok、Reddit、X、小红书（海外版）、Pinterest。 |
| D5 | 履约方案（验证通过后） | 阿里巴巴海外仓小批量发货；豆子包 = 按图纸精确数豆分装 + 板子 + 烫纸。 |
| D6 | 核心承诺 | 最完整的服务、最及时的响应、**色卡数据最准、拼完不剩豆、不缺豆**。注意：「不剩豆不缺豆」只有自营精确分装套装能兑现，V1 阶段作为套装预售的卖点而非全站承诺。 |
| D7 | 色卡锚定（默认值，可否决） | **Perler 为 V1 默认色卡**（美国用户存量最大、亚马逊联盟链接可直接闭环）；Artkal 作为未来自营套装的供货体系（国内易采购）预留第二套色卡。两套色号体系互不兼容，数据结构按多品牌设计。 |
| D8 | 电商启动门槛（建议值） | 上线后 30 天内预售意向 ≥ 50 单（或预售按钮点击→留资转化率 ≥ 8%），则启动海外仓自营套装；否则继续联盟链接模式并迭代内容。 |

## 4. 目标用户与痛点

| 用户 | 特征 | 痛点 | 我们的解法 |
|------|------|------|-----------|
| 新手（主力） | 被社媒内容种草，从未玩过 | 完全不知道要买什么 | 科普页 + 图纸自带完整购物清单（豆、板、纸、熨斗） |
| 玩过 1~2 次 | 家里有一盒混色豆 | 色号不全、凑不齐，买整袋剩一堆 | 精确到颗的 BOM；（未来）缺色补齐包 |
| 痴迷玩家（少数） | 自己会找图做图 | 想要更准的色卡和更快的出图 | 高精度转换器留住他们，贡献 UGC 与口碑 |

关键洞察：**除痴迷玩家外，用户不会主动来，全部靠内容触发**。消费由「看到一条爆款
视频/图文」驱动，所以图纸库和营销素材是同一份工作。

```

```

## 6. V1 功能范围（MoSCoW）

### Must（V1 上线必须有）
1. **转换器（Studio）**：上传图片 → 选板子尺寸（29×29 / 50×50 / 58×58）→ 选最大用色数
   → 生成带编号的拼豆图纸（每格标注色号编号）。纯客户端实现，无服务端成本。
2. **Perler 色卡映射**：Lab 色彩空间最近色匹配；色数超限时按使用频次向最近色合并。
3. **BOM 清单**：每个色号的名称、色块、**精确颗数**、亚马逊联盟购买链接；合计豆数。
4. **图纸导出**：高清 PNG 图纸下载（含网格、编号、图例、颗数）+ 浏览器打印。
5. **$19.9 预售按钮**：套装内容说明（精确分装豆子 + 板 + 烫纸），收集购买意向
   （V1 用邮件留资，V1.1 换 Stripe Payment Link 预付）。
6. **新手科普区**：三步教程 + 入门装备清单（联盟链接）。
7. **落地页**：Q 版叙事第一屏 + 示例图纸展示。

### Should（V1.1，上线后 2~4 周）
- AI Q 版化（接 gpt-image 或 Nano Banana API，服务端代理，控制成本 ~$0.02-0.04/次）。
  **计费时刻**：生成与预览免费（低清/带水印），在「下载高清图纸 + BOM」的那一刻收费
  ——付费墙摆在用户已喜欢结果之后（拼豆星球实证：保存扣 1 豆 = ¥2.8/张，注册赠 1 豆
  即免费完整体验一次）。海外定价 $0.99/张 或订阅（无限生成 + 免广告 + 免费下载打包）；
  免费额度 = 首张完整免费。保留「每次生成结果不同，可重试」机制，但首次含 2 次免费
  重试（比竞品体面，利口碑）。风格命名只用 Toon / Chibi / Big-head 等通用词，
  不碰品牌词（竞品用「迪士尼」是海外不可复制的反例）。
- 原创 Q 版图纸库（每张 = 一条社媒素材），「适合 XX 粉丝的配色」话题合集页。
- Stripe 预售真实收款；Plausible/GA 漏斗埋点。
- Artkal 色卡（对接自营套装供应链）。
- 图纸 ID 体系 + 回流水印：导出图纸页脚印 beadfable.com/p/{id}，
  每张被分享/打印的图纸都是带回流入口的广告（抄拼豆星球「搜索【编号】」机制）。
- 镜像导出（熨烫后翻面比对用）。

### Won't（V1 明确不做）
- 社区/UGC 平台、评论、账号体系
- 自营电商与库存（等 D8 门槛验证）
- 移动 App、多语言
- 万能风格转换器叙事（引擎保留，叙事只讲 Q 版）
- 真人明星 Q 版套装（形象权风险，见 §8）
- 虚拟货币/任务返利体系（竞品的豆子经济是人力黑洞，小团队不可复制；
  海外合规版 share-for-discount 排 V2）

### V2 候选（记录，不排期）
- **拼豆模式**：逐格引导拼豆的互动模式（竞品已验证的游戏化留存功能）
- 像素编辑器（改格/换色/实时用量统计）
- 竞品完整拆解见 docs/competitor-pindou-planet.md

## 7. 变现模型与单位经济测算

| 收入流 | V1 状态 | 量级判断 |
|--------|---------|----------|
| 亚马逊联盟佣金 | 上线即有 | 玩具类 ~3%，单均几十美分，仅覆盖成本，本质是需求探针 |
| 豆子包预售 $19.9 | 意向验证 | **核心验证对象** |
| 图纸付费/订阅 | 不做 | 竞品证明天花板矮，V1 图纸免费换流量 |

**$19.9 套装成本倒算（假设，待采购报价校准）**：
豆子+分装人工 ~$2.5，板+烫纸 ~$1.5，包材 ~$0.5，海外仓头程摊销 ~$1，尾程配送 ~$4.5，
支付手续费 ~$0.9 → 成本合计 ~$10.9 → **毛利 ~$9 / 毛利率 ~45%**。
若毛利率 < 35% 则调价至 $24.9 或做双图纸套装提客单价。

## 8. 内容营销计划与 IP 合规红线

### 内容策略
- 渠道优先级：TikTok（过程视频）> Pinterest（图纸长图）> Reddit r/beadsprites 等
  （教程/成品）> X、小红书海外。
- 素材 = 图纸库：每张原创 Q 版图纸同时产出「成品图 + 过程视频 + 图纸长图」三件素材。
- 话题挂钩：蹭热门影视/游戏/人物**话题**（如 "perfect palette for XX fans"），
  内容展示的是**原创角色与配色**。

### IP 红线（不可越）
1. ❌ 网站不出售、不预制任何受版权保护角色（任天堂、迪士尼等）的图纸或套装。
2. ❌ 不以真人明星肖像制作、销售套装（美国 right of publicity 风险）。
3. ✅ 用户自行上传的图片由用户负责（服务条款声明 + DMCA 响应流程）。
4. ✅ 营销可参与话题讨论，但素材使用原创设计。

## 9. 技术方案

- **部署**：Cloudflare Workers 静态资产（`wrangler.jsonc` assets 模式），零服务器成本。
- **转换引擎**（纯客户端 `public/app.js`）：
  Canvas 降采样（cover 裁切到 N×N）→ sRGB→Lab → CIE76 最近色映射到色卡 →
  超出最大用色数时，按使用频次将最少色合并到最近的保留色 → 网格渲染（每格编号）。
- **色卡数据**（`public/palettes.js`）：多品牌结构 `{ brand: { name, colors: [{code, name, hex}] } }`；
  V1 内置 Perler 常用 43 色。
- **导出**：Canvas 合成完整图纸页（标题 + 网格 + 图例 + 颗数）→ PNG 下载 / 打印。
- **AI Q 版化（V1.1）**：Worker 路由代理图像 API（密钥存 Workers Secret），前端不暴露密钥。
- **预售留资（V1）**：mailto 预填模板；V1.1 换 Stripe Payment Link + Workers KV 存单。

### 色卡数据策略（兑现「色卡最准」）
V1 内置 hex 为社区通用近似值，**上线前必须做一次实物校准**：购入 Perler 全色号，
标准光源下拍照取色，修正 `palettes.js`。这是「色卡数据最准」承诺的实际工程，
也是竞品少有人做的脏活（差异化点）。
