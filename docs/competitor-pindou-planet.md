# 竞品拆解：拼豆星球（微信小程序）

- 日期：2026-07-04
- 素材：10 张实拍截图（两批），覆盖首页/赚豆子/创作中心/AI 风格化/作品详情/编辑器/导出
- 结论供 BeadFable PRD（docs/PRD-V1.md）引用

---

## 1. 产品架构

五 Tab：首页（关注/笔记/图纸市场）· 赚豆子（任务 + 商城）· 做图纸（创作中心）·
消息 · 我的。本质是**以「豆子」虚拟货币为血液的 UGC 社区电商**，工具退居第三 Tab。

## 2. 变现结构（按重要性）

| 变现点 | 形态 | 观察到的定价 | 备注 |
|---|---|---|---|
| **AI 风格化生成** | 豆子按次扣费 + 看广告预览(3次) + 会员免广告 | **¥2.8/张** | API 成本约 $0.02-0.04，毛利 90%+；「同一风格每次生成都会变化，可多次尝试」= 抽卡式重复付费 |
| **按图配豆** | 商城 SKU「根据图纸所需豆子数量发货（无需购买全套221）」 | **¥19.9 起** | 与 BeadFable 套装假设同构；国内 ¥19.9 → 海外 $19.9 即出海溢价空间 |
| 全套色号/标品 | 商城 | ¥9.9 级 | 标品，拼价格 |
| 图纸定制 | 人工手绘服务 | 未知（高客单） | 服务型长尾 |
| UGC 图纸售卖 | 免费/广告解锁/付费 | ¥0.5/张 量级 | 图纸本身不值钱，验证 PRD「图纸是引流品」判断 |
| 会员 | 免广告不限次预览 | 未知 | |

## 3. 增长引擎（全部众包）

- 小红书/淘宝**好评返豆**（+5 豆）：用虚拟货币收买真实买家秀，养自己店铺评价。
- 发抖音/小红书/快手提及小程序 +1~10 豆；邀请注册 +3 豆；精选图纸投稿 +3 豆。
- **导出图纸自带水印**：「打开拼豆星球，搜索【图纸编号】获得超清原图」——
  每张被分享/打印的图纸都是带回流入口的广告。图纸编号 = 搜索码 = 拉新链路。
- 作品详情页红字「获得方式：未发布广场」催发布 → 供给侧靠 UGC，官方饥渴。

## 4. 核心链路细节（第二批截图）

- **AI 风格化**：8 风格（大头Q版/小图Q版/迪士尼/原图/高度还原/Chibi/猫/车）+
  自定义文本要求（如「衣服改成红色、背景更简洁」）。
  ⚠️ 风格直接命名「迪士尼」——海外不可复制，BeadFable 用 Toon/Chibi/Big-head 等通用词。
- **作品详情**：大小/色号品牌(mard)/色号数量/浏览/使用 元数据规范；开始拼豆按钮。
- **编辑模式**：完整像素编辑器（画笔/橡皮/油漆桶/取色/选区/镜像/撤销重做），
  底部色板实时显示每色号用豆数。
- **拼豆模式**：逐格引导拼豆的互动模式（= 拼豆「游戏」的实际形态，留存功能）。
- **导出配置**：色号开关/统计/镜像（熨烫翻面用）/网格深浅/行列坐标/底色/辅助线，
  服务端渲染（「根据选项调用接口生成图片」）。
- **成品图纸**：每格印色号缩写、四边行列坐标、底部图例条含每色用量、
  作者署名、色号数/豆子数合计、回流水印。

## 5. 它的软肋（= BeadFable 的窗口）

1. **默认参数坑新手**：实测照片直转 104×104 / 221 色全开 → 56 色、10,816 颗、
   效果糟。复杂度全丢给用户。BeadFable 反着做：强预设（29×29、≤12 色）、
   Q 版单一叙事、出来就是拼得完的东西。
2. **IP 风险内嵌**：付费风格叫「迪士尼」、UGC 公开售卖蜡笔小新图纸。国内灰色，
   海外致命。BeadFable 红线见 PRD §8。
3. **锁死在微信小程序 + 漫漫(mard)色号 + 中文世界**：出不了海的三道墙。
4. **重运营**：豆子经济、任务审核、UGC 审核都是人力黑洞；单人/小团队不可复制，
   也不必复制。
