## 1. 产品概述

### 1.1 一句话定位

**面向海外拼豆爱好者的「Q 版拼豆图纸 + 精确到颗的物料清单」内容交易站**：
社媒话题内容引流 → 网站承接转化 → AI 图纸变现 + 联盟链接 + 精确配豆套装。

### 1.2 商业逻辑（三层递进）

1. **这不是工具站**。纯转换工具赛道已有 10+ 竞品，付费天花板 $14.99/年量级。
   工具免费，是获客入口和信任建立器。
2. **这是话题营销驱动的交易站**。用户不会主动来——除痴迷玩家外，消费全部由
   「在 TikTok/小红书/Pinterest 看到一条爆款内容」触发。图纸库和营销素材是同一份工作。
3. **收银台有两个**：即时的（AI 图纸下载费/订阅，毛利 ~95%）和大额的
   （$19.9 精确配豆套装，毛利 ~45%，出海溢价 5~7 倍的载体）。

### 1.3 目标市场

海外英语区（首站美国），女性向手工/减压人群。渠道：TikTok、Pinterest、
Reddit（r/beadsprites 等）、X、Instagram。

### 1.4 核心承诺

最完整的交付、最及时的响应、色卡数据最准、**拼完不剩豆、不缺豆**
（后者由精确分装套装兑现，是全站差异化的锚）。

## 2. 市场与竞争

### 2.1 市场规模

- 全球熔融豆市场 ~6.6 亿美元（2026），预计 2035 年 10.7 亿美元。
- 海外成熟市场三分天下：Perler（~24% 份额，美国）、Hama（欧洲）、Artkal（中国出海）。
- 中国供给端过热：义乌日发货 ~25 亿颗；淘宝拼豆搜索量同比 +500%（2025）。
- **价差即机会**：国内 ¥39 的货海外卖 $27，5~7 倍溢价；竞品「按图配豆」国内收
  ¥19.9，同等履约海外可收 $19.9。

### 2.2 产业链利润分布

| 环节 | 特征 | 我们的位置 |
|---|---|---|
| 制造（义乌） | 薄利走量 | 不做，采购即可 |
| 品牌（Perler/Hama/Artkal） | 品牌溢价 | 不做 |
| **跨境零售/套装** | 5~7 倍溢价 | ✅ D8 验证后进入 |
| 线下体验店 | 现金流好、重运营 | 不做 |
| 图纸/工具 | 毛利高、天花板矮 | ✅ 免费引流 + AI 增值收费 |

### 2.3 竞争格局

**海外工具站**（beadpattern.net、Somake、PixelPerler、pixelbeads.org、Beadify 等）：
共同断点是停在「导出图纸」，无人做「按图配料」闭环；无人做「拼得完」的预设收敛。

**拼豆星球**（国内标杆，完整拆解见 docs/competitor-pindou-planet.md）——五环闭环：

```
广告 + 按次保存费（¥2.8/张）养 AI 工具
  → 工具产出图纸（UGC 化，全量人工审核）
    → 「按图配豆」实物订单（¥19.9 起）
      → 豆子任务驱动社媒拉新 + 好评（零成本市场预算）
        → 私域（个人微信 + 群）承接复购
```

它出不了海的三道墙：微信小程序生态、漫漫（mard）色号体系、中文世界。
它可被利用的软肋：默认 221 色全开坑新手（实测 104×104 转出 56 色 10,816 颗的
不可完成品）；「迪士尼」风格命名与 UGC 售卖侵权图纸在海外是致命伤。

### 2.4 BeadFable 对位

| 拼豆星球闭环 | BeadFable 对位 | 阶段 |
|---|---|---|
| AI 工具 + 保存收费 | AI Chibi Maker，下载时刻收费 | V1.1 |
| 图纸供给（UGC+审核） | 官方原创图纸库（无 UGC，零审核负担） | V1.1 |
| 按图配豆 ¥19.9 | 精确配豆套装 $19.9（预售验证中） | V1 → D8 |
| 豆子任务营销 | 海外合规版：分享得折扣 | V2 |
| 个人微信私域 | 邮件列表起步，寄生 Reddit/Pinterest | V1 起步 |

## 3. 用户

| 用户 | 特征 | 痛点 | 解法 |
|---|---|---|---|
| 新手（主力） | 被社媒内容种草，从未玩过 | 完全不知道要买什么；怕做不完 | 强预设「一晚拼完」+ 图纸自带完整购物清单 |
| 玩过 1~2 次 | 家里有一盒混色豆 | 色号不全、凑不齐；买整袋剩一堆 | 精确到颗 BOM；（D8 后）缺色补齐包 |
| 痴迷玩家（少数） | 自己会找图做图 | 想要更准色卡、更快出图 | 高精度引擎 + 58×58/24 色上限留住，贡献口碑 |

关键洞察：**消费由内容触发，不由需求搜索触发**。落地页的对话对象永远是
「刚看完一条视频、拿着手机跃跃欲试的人」。

## 5. 范围与路线图

### 5.1 V1（已建成，2026-07-04 实测通过）

1. **Pattern Studio**：上传图片 → 板尺寸（29×29 默认 / 50×50 / 58×58）→
   最大用色数（4~24，默认 12）→ 带编号图纸。纯客户端，零服务器成本。
2. **Perler 色卡映射**：43 色，Lab 空间 CIE76 最近色；超预算按使用频次向最近色合并。
3. **BOM**：每色号名称/色块/精确颗数/亚马逊联盟链接；合计豆数与色数。
4. **导出**：高清 PNG 图纸页（标题 + 编号网格 + 每 5 格行列坐标 + 图例用量）+ 打印。
5. **$19.9 套装预售按钮**：弹窗说明套装内容，邮件留资（自动附带该图纸色号清单）。
6. **新手科普区**：三步教程 + 入门装备联盟链接。
7. **落地页**：Q 版第一屏 + 「Made to be finished, not abandoned」对比区
   （典型转换器 10,000+ 颗灾难 vs BeadFable 一晚拼完）+ 隐私说明（纯浏览器处理）。

### 5.2 V1.1（上线后 2~4 周）

| 功能 | 规格要点 |
|---|---|
| **AI Chibi Maker** | 服务端代理 gpt-image / Nano Banana（密钥存 Workers Secret）；4~6 个通用命名风格（Chibi / Big-head / Toon / Faithful）；自定义文本要求输入框；按 D9 计费；成本 ~$0.02-0.04/次，售 $0.99/张 |
| **订阅** | 打包结构抄竞品：无限生成 + 免广告位 + 免费下载高清；定价提案 $4.99/月 或 $29/年（上线前 A/B） |
| **Stripe** | 套装预售 Payment Link（真实预付替代邮件留资）+ AI 下载单次收款 |
| **图纸 ID + 回流水印** | 每张导出图纸页脚印 beadfable.com/p/{id}；分享即广告（抄竞品搜索码机制） |
| **镜像导出** | 熨烫翻面比对用 |
| **原创图纸库** | 首批 10 张原创 Q 版（每张 = 成品图 + 过程视频 + 图纸长图三件套社媒素材）；「适合 XX 粉丝的配色」话题合集页 |
| **邮件订阅** | 落地页 + 导出成功页收邮箱（私域第一块砖，D11） |
| **埋点** | Plausible（漏斗见 §9） |
| **Artkal 色卡** | 第二品牌色卡上线（自营套装供货体系） |
| **Perler 色卡实物校准** | 购入全色号，标准光源拍照取色修正 palettes.js（「色卡最准」的工程兑现） |

### 5.3 V2 候选（不排期，D8 通过或 V1.1 数据支撑后评估）

- **自营精确配豆套装**（D8 门槛触发）：阿里海外仓履约，Artkal 供货
- **拼豆模式**：逐格引导拼豆的互动模式（竞品已验证的游戏化留存）
- 像素编辑器（改格/换色/实时用量）
- 分享得折扣（豆子任务的海外合规版）
- Discord 社群；缺色补齐包（复购型 SKU）

### 5.4 Won't（明确不做）

- UGC 图纸广场、虚拟货币、任务返利（D10）
- 真人明星 Q 版套装（形象权风险）
- 万能风格转换器叙事（引擎保留，叙事只讲 Q 版）
- 移动 App、多语言（英文单语先行）
- 微信生态任何形态

## 6. 核心用户旅程

```
社媒内容（原创 Q 版爆款 + 制作过程视频，蹭话题不蹭形象）
  → 落地页（同款图纸详情 / 上传自己的照片）
    → Studio：上传 → AI Q 版化（V1.1）→ 像素化 → Perler 色卡映射
      → 免费：图纸预览 + BOM
        → 变现 A：BOM 行「Buy on Amazon」联盟链接（即时、小额）
        → 变现 B：AI 高清图纸下载 $0.99 / 订阅（V1.1，即时、高毛利）
        → 变现 C：$19.9 精确配豆套装（预售→自营，大额、差异化锚）
          → 导出图纸带回流水印 → 被分享/打印 → 新用户（病毒环）
          → 邮箱入库 → 邮件触达复购（私域环）
```

## 7. 商业模式与单位经济

### 7.1 收入流矩阵

| 收入流 | 阶段 | 单价 | 边际成本 | 毛利 | 角色 |
|---|---|---|---|---|---|
| 亚马逊联盟 | V1 | 订单额 ~3% | 0 | 100% | 需求探针，覆盖成本 |
| AI 图纸下载 | V1.1 | $0.99/张 | ~$0.03 | ~95% | 现金流主力（竞品实证 ¥2.8/张成立） |
| 订阅 | V1.1 | $4.99/月（提案） | ~生成用量 | 高 | 重度用户买断摩擦 |
| **精确配豆套装** | 预售→D8 | $19.9 | ~$10.9 | ~45% | 差异化锚 + 客单价主力 |
| 图纸单卖 | 不做 | — | — | — | 竞品 ¥0.5/张证明不值得，图纸免费换流量 |

### 7.2 套装成本倒算（假设值，采购报价后校准）

豆子+分装人工 ~$2.5，板+烫纸 ~$1.5，包材 ~$0.5，海外仓头程摊销 ~$1.0，
尾程配送 ~$4.5，支付手续费 ~$0.9 → **成本 ~$10.9，毛利 ~$9.0（45%）**。
毛利率 < 35% 时调价 $24.9 或双图纸套装提客单。

### 7.3 免费额度设计（D9）

注册/首访即可完整体验一次（首张 AI 图纸含高清下载全免费）→ 第二张起付费。
竞品用「注册赠 1 豆」验证了「免费尝一次完整价值」的激活效果。

## 8. 增长、内容与合规

### 8.1 内容策略

- 渠道优先级：TikTok（过程视频）> Pinterest（图纸长图，长尾搜索流量）>
  Reddit（教程/成品，社区信任）> Instagram / X。
- **素材 = 图纸库**：每张原创图纸产出三件素材（成品图 / 过程视频 / 图纸长图）。
- 话题挂钩：蹭热门影视/游戏/人物**话题**（"perfect palette for XX fans"），
  展示的永远是原创角色与配色。

### 8.2 病毒环与私域（零预算增长）

- 导出图纸自带 beadfable.com/p/{id} 水印（V1.1）——每张被打印/分享的图纸是广告。
- 预售与下载环节收邮箱 → 邮件列表 → 新图纸/套装上新触达。
- 不自建社群，先寄生 Reddit/Pinterest 已有氛围（D11）。

### 8.3 合规红线（不可越）

1. ❌ 不出售、不预制受版权保护角色的图纸或套装（任天堂/迪士尼为高危）。
2. ❌ 不以真人明星肖像制作销售套装（right of publicity）。
3. ❌ 不做激励性好评/返利换评价（亚马逊与 FTC 红线；竞品的返豆模式不可搬）。
4. ❌ AI 风格不用品牌词命名（竞品「迪士尼」风格是反例）。
5. ✅ 用户上传自担责：ToS 声明 + DMCA 响应流程（V1.1 上线前挂页）。
6. ✅ 联盟链接页面披露（已上线："As an Amazon Associate we earn…"）。

## 10. 技术架构

### 10.1 现状（V1，已上线代码）

- **部署**：Cloudflare Workers 静态资产（`wrangler.jsonc` assets 模式），零服务器成本。
- **引擎**（`public/app.js`，纯客户端）：cover 裁切降采样 → sRGB→Lab →
  CIE76 最近色映射 → 色数预算合并（最少用色并入最近保留色）→
  编号网格渲染（每 10 格粗线）→ BOM 统计 → Canvas 合成图纸页
  （标题/网格/每 5 格坐标/图例用量）导出 PNG/打印。
- **色卡**（`public/palettes.js`）：多品牌结构；Perler 43 色（社区近似值，待实物校准）。
- 隐私卖点：图片不出浏览器（落地页已声明）。

### 10.2 V1.1 增量

- Worker 路由 `/api/stylize`：代理图像生成 API，密钥存 Workers Secret，
  按 IP/session 限额（免费额度与防滥用）。
- Workers KV：图纸 ID → 元数据（水印回流页 `/p/{id}` 用）；预售订单记录。
- Stripe：Payment Link（套装预售）+ Checkout（AI 单次下载）+ webhook 落 KV。
- Plausible 脚本 + 自定义事件（§9 漏斗）。

### 10.3 色卡数据策略

V1 hex 为社区近似值 → V1.1 实物校准（全色号标准光源拍照取色）→
页面保留 "screen colors are approximate" 声明直至校准完成。
这是「色卡最准」承诺的实际工程，也是竞品少有人做的脏活。
