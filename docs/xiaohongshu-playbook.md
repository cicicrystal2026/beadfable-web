## 2. 核心认知：痛点 / 痒点 / 爽点

- 痛点是恐惧（让人逃跑）；**痒点是想成为更理想的自己（让人掏钱）**；
  爽点是即时满足（付款立刻交付）。
- 用户买的不是图纸，是**身份切换的开关**——付款那一刻她就是"心灵手巧的人"，
  拿图之后做不做不重要。**你卖的是那条最短的路。**
- 与拼豆星球「保存收费」是同一逻辑的不同载体：都把收费摆在
  「情感投入完成、价值交割」的瞬间。

## 3. 下单链路与可复制素材

```
爆款笔记（before/after 对比图 + 痒点文案）
  → 店铺虚拟商品（¥4.9~9.9 自动发货） / 引流私域（原作者最推荐）
    → 25% 分销佣金放大铺量
```

- **商品标题公式**：照片转拼豆图纸（有色号）· 自助工具 · 自动发货 · 不满意可退
  ——「有色号」是专业信号词；「自动发货+可退」是信任三件套。
- **笔记标题**：🔥拼豆人狂喜！这个AI工具让你实现"图纸自由"
- **正文开头**：宝子们，谁懂啊！每次做拼豆最头疼的就是找图纸、画网格……
  直到我发现了这个宝藏AI工具（痒点叙事，讲身份不讲功能）。
- **商品图**：AI 生成 before/after（宠物照 → 拼豆图）+ 大字「一键生成」「无限次转换」。
- **定价错位**：竞品 ¥4.9/张（或 3 张/10 张次数包），可卖 ¥9.9 无限次——
  同市场用计费单位差异化。
- **明确不做**：小程序（成本高，笔记+店铺足够）。
- **过程即内容**：建站/做图过程录屏本身就是一篇引流笔记。
